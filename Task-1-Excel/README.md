°Project Overview°

 In this task, I used Microsoft Excel to explore and analyze the dataset in order to better understand the information it contains. The first step was to review the data and organize it properly so that it could be analyzed efficiently.

  To make the results easier to interpret, I also created charts and visual representations of the data. This task helped me strengthen my ability to use Excel not only for data organization but also for performing basic analytical work and presenting insights clearly.
